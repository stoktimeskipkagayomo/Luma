服务器基础信息

API 地址： http://localhost:5102
当前状态： ✅ 运行中
协议版本： OpenAI API v1 兼容

🤖 核心 AI 接口

POST /v1/chat/completions
用途：与 AI 模型对话
支持：流式/非流式、多模态消息、文生图
GET /v1/models
用途：获取可用模型列表
返回：所有配置的模型信息

📊 监控与统计

GET /monitor
实时监控面板（当前页面）
GET /api/monitor/stats
获取实时统计数据（JSON）
GET /api/monitor/active
查看当前活跃请求

📈 日志与详情

GET /api/request/{request_id}
获取请求完整详情
GET /api/monitor/logs/requests
参数：limit=50（默认）
返回：最近的请求日志
GET /api/logs/download
参数：log_type=requests|errors
返回：JSONL格式日志文件

🔌 WebSocket 连接

WS /ws
用途：油猴脚本通信
协议：双向消息传递
WS /ws/monitor
用途：监控面板实时更新
数据：统计、活跃请求、事件
